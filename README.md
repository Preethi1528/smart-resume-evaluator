# Smart Resume Evaluator

This project evaluates resumes and provides insights on skills, experience, and resume quality.

## Folder Structure

- `smart resume frontend/` - Frontend of the application (HTML, CSS, JS)
- `smart resume backend/` - Backend logic (Python)

## How to Run

1. Set up Python environment for backend.
2. Run the backend server.
3. Open `frontend/index.html` in a browser.
